8. Prop-types
9. Reactive Search

# How to use
1. Install Node.js
2. Run cmd in project folder
3. Copy and paste commands below to install modules:

```
npm install react-app-rewired --save-dev && npm install 
```

4. Now everytime you want to run website, just type `npm start` in cmd
